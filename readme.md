## 4. apache benchmark
--------

### 4.1 ab: command 
```bash
ab -p post_json.txt -T application/json -n 10000 -c 1000 -k 'http://192.168.10.200:5002/md'
```

+ n: sum
+ c: batch
+ k: keep-alive
+ T: special json
+ p: post request

### 4.2 post file format

```bash
hugo@ubuntu-server:~$ cat post_json.txt 
{"symbols" : ["EUR_USD","CHF_HKD","CHF_HKD"]}
```
